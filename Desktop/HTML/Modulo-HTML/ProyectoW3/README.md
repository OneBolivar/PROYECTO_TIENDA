**TECNOLOGIAS USADAS**
- HTML5: Estructura semántica del contenido.
- CSS3: Estilos avanzados y layouts.
- Sass: Preprocesador para una arquitectura de estilos modular y mantenible.
- JavaScript : Lógica del lado del cliente e interactividad.
------------------------------------------------------------------------
**CARACTERISTICAS**
- Diseño **Mobile First** y totalmente responsivo.
- Estilos modulares organizados con **Sass**.
- Manipulación dinámica del DOM mediante **JavaScript**.

**Instalación y Uso Local**
Si quieres clonar este proyecto y ejecutarlo en tu máquina local, sigue estos pasos:
1.**Clonar el repositorio:**
   ```bash
   git clone https://github.com
   ```

2.**Entrar a la carpeta:**
   ```bash
   cd nombre-del-proyecto
   ```

3.**Compilar Sass (si es necesario):**
   Si no usas la extensión *Live Sass Compiler* de VS Code, ejecuta:
   ```bash
   sass --watch scss/main.scss:css/style.css
   ```

4.**Abrir el proyecto:**
   Simplemente abre el archivo `index.html` en tu navegador favorito.
